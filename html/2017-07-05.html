@stuartgpalmer Reflection field 2.0.0 introduced some breaking changes, but also some bugs and quirks in the way xpath-expressions were processed. Some of these were fixed in 2.0.2 but these fixes came with other breaking changes (and bugs) regarding xpath-expressions.
