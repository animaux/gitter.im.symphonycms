michael-e
@michael-e
Jan 13 15:27
Create PDFs from XSL-FO. I heard that XSLT was only invented to help creating XSL-FO files. (At least it is sure that XSL-FO is older than XSLT.) Since Symphony deals so much with XML, it feels pretty "natural" to describe print documents with XML (which is the rationale behind XSL-FO).
You need Java on the server in order to run Apache FOP (which, unfortunately, is the only Open Source FO processor).
