You can even have unlimited private repos in the free version. It's just some features that are added if you go for a paid plan. However, I still don't fully understand what is meant by a "user" in paid plans.
You don't need a how-to to move a GitHub repo. Just click on "New project", then "Import project". After choosing GitHub, you will have to provide GitHub permissions. I chose the "Access Token" alternative, because I knew that I can simply remove the token from my GitHub account afterwards. (Permissions are read and write, so I wanted to be extra-careful.)
